## 🚀 Tecnologías del Proyecto (Este Repo)

Para lograr la estética de **Terminal Emulator** y asegurar un despliegue eficiente, utilicé el siguiente stack:

* **JavaScript (Vanilla):** Motor lógico del sitio para el procesamiento de comandos e historial, priorizando la velocidad de respuesta.
* **HTML5 & CSS3:** Diseño basado en variables CSS para replicar el efecto "CRT/Scanline" y estética hacker.
* **GitHub Pages:** Hosting estático para el servicio del portafolio.
* **GitHub Actions:** Pipeline de CI/CD para despliegue automático.
